## ローカルテスト方法

### 必要な環境

- Ruby
- Bundler
- Jekyll

### 手順

1. 上記必要なコンポーネントをインストールする
2. `bundle install` を実行する
  - VS Code では `bundle-install` タスクから実行可能
3. `bundle exec jekyll serve` を実行する
  - VS Code では「実行とデバッグ」から実行可能
